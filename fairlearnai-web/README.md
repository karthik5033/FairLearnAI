# FairLearnAI - Web Application\n\nAI-powered educational platform with quiz generation, teacher analytics, and gamification.\n\n## Tech Stack\n- Next.js 15\n- TypeScript\n- Tailwind CSS\n- Framer Motion\n- Google Gemini API\n\n## Getting Started\n\n1. Install dependencies:\n`ash\nnpm install\n`\n\n2. Set up environment variables:\nCreate a .env.local file with:\n`\nGEMINI_API_KEY=your_api_key_here\n`\n\n3. Run development server:\n`ash\nnpm run dev\n`\n\n## Features\n- Student Dashboard with gamification\n- Teacher Dashboard with analytics\n- AI-powered quiz generation\n- Real-time progress tracking
